### ЁЯОп Why this topic matters

Every Linux Administrator, DevOps Engineer, Cloud Engineer, and SRE must understand the Linux file system.

If you know where files live, troubleshooting becomes easier:

- Configuration issue? Check `/etc`
- Log issue? Check `/var/log`
- User issue? Check `/home`
- Process issue? Check `/proc`
- Disk issue? Use `df` and `du`
